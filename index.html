Introduction

JavaScript est un langage de programmation interprété, utilisé principalement pour rendre les pages web interactives. Il fonctionne côté client (navigateur) et peut également être utilisé côté serveur grâce à Node.js.

What you should to know about JS?

Avant de commencer à apprendre JavaScript, il est utile de maîtriser certains concepts de base pour faciliter ton apprentissage. Voici une liste des connaissances préalables recommandées :

  1. Notions de base en HTML et CSS

    2. JavaScript est étroitement lié à HTML et CSS pour rendre les pages web interactives. Il est donc essentiel de comprendre les bases de ces deux technologies.

    HTML (HyperText Markup Language) :
    CSS (Cascading Style Sheets) :
    • Savoir appliquer des styles aux éléments HTML (propriétés comme color, font-size, margin, padding).
    • Comprendre comment utiliser les classes (.class) et les IDs (#id) pour styliser des éléments.
    • Connaître les concepts de base de mise en page comme Flexbox et Grid.
    1. Notions de base en programmation

    Si tu as déjà appris un autre langage de programmation (comme Python ou C++), beaucoup de concepts seront similaires.

    Concepts essentiels:
    • Variables: Savoir comment déclarer et utiliser des variables pour stocker des données.
    • Types de données: Comprendre les types comme number, string, boolean, array, et object.
    • Conditions: Utiliser if, else if, et else pour la prise de décisions.
    • Boucles: Savoir répéter des tâches avec des boucles comme for, while, ou do-while.
    • Fonctions: Écrire des blocs de code réutilisables.
    example: function saluer(nom) { return `Bonjour, ${nom}!`; } console.log(saluer("Imen")); // Affiche : Bonjour, Imen!
    1. concepts informatiques de base
    • Comment fonctionne le web : Comprendre les bases du fonctionnement des navigateurs, des serveurs, et des requêtes HTTP.
    • Structure des fichiers : Savoir organiser un projet avec des fichiers séparés (HTML, CSS et JavaScript).
    • Débogage : Avoir des bases en dépannage des erreurs en utilisant les outils de développement du navigateur (ex. : Chrome DevTools).
    1. Notions de base sur la ligne de commande

      2. Ce n'est pas obligatoire, mais savoir utiliser la ligne de commande est un atout pour :

      • Naviguer dans les fichiers et dossiers (cd, ls).
      • Exécuter des commandes pour initialiser des projets (par exemple, npm init pour les projets JavaScript).

JavaScript and Java

JavaScript et Java sont deux langages de programmation complètement différents, malgré leurs noms similaires. Voici une comparaison claire pour bien comprendre leurs différences :

  1. Origine et conception :
Critère JavaScript Java
Création 1995 par Brendan Eich chez Netscape 1995 par James Gosling chez Sun Microsystems
But initial Rendre les pages web interactives (langage côté client). Développement d'applications multiplateformes.
Nom Le nom "JavaScript" a été choisi pour surfer sur la popularité de Java, mais ils n’ont aucun lien technique. Inspiré du langage C++ pour les fonctionnalités et le style.
